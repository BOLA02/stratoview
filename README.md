## 🎯 Project Overview

### Objective
To build a premium, highly visible digital footprint that showcases SNL’s technical expertise, drives B2B project inquiries, and serves as a verified asset during proposal and tender submissions.

### Core Features
- 📱 **Responsive & Mobile-First:** Fluid layouts optimized across all screen formats.
- 🔒 **Enterprise Security:** SSL encryption protocol throughout the site.
- 💬 **Instant Communication:** Direct WhatsApp Business API integration.
- 📰 **Thought Leadership:** Dedicated Blog/Insights matrix to post industry papers.
- 📁 **Asset Downloads:** Secure, downloadable links for the Corporate Profile and Capability Statements.

### Advanced Features
- 📝 **RFx Gateway:** Built-in Request for Proposal (RFP) submission form with secure file uploads.
- 💼 **Talent Pipeline:** Careers module featuring current vacancies and CV routing.
- 📊 **Analytics Pipeline:** Native tracking hookups for marketing attribution.

## 🗺️ Information Architecture & Site Structure

### 1. Home Page
* **Hero Canvas:** 
  * *Headline:* "Geoscience, Environmental & Sustainability Solutions for Responsible Development"
  * *Subheadline:* "Helping organizations make informed decisions through environmental consulting, mineral exploration, geotechnical investigations, hydrogeology, GIS services, and technical advisory."
  * *CTAs:* `[Request Consultation]` | `[Download Company Profile]`
* **Sections:** Executive Summary $\rightarrow$ Services Overview $\rightarrow$ Industries Served $\rightarrow$ Value Proposition $\rightarrow$ Selected Portfolio Case Studies $\rightarrow$ Insights feed $\rightarrow$ Trust Testimonials $\rightarrow$ Site-wide Contact Grid.

### 2. About Us
* **Company Overview:** History, background operational framework, and corporate philosophy.

* **Mission:** To provide innovative, data-driven, and environmentally responsible consulting services that support sustainable development.

* **GIS & Spatial Intelligence:** GIS Mapping, Remote Sensing, Spatial Data Analytics, Environmental Threat Mapping.

## 🛠️ Technical Architecture

### Stack Selection
* **Recommended Engine:** WordPress CMS (Engineered with low-overhead block editors for long-term scalability and fluid updates by non-technical staff).
* **Alternative Option:** Webflow (For advanced visual canvas management and premium animations).
* **Hosting Paradigm:** Cloud infrastructure engineered with automated edge daily backups, enterprise-tier SSL termination, and a high-availability CDN layer.

### Core Engineering Requirements
* Extreme loading optimization (<2.5s LCP benchmark).
* Accessible semantic HTML elements compliant with WCAG design standards.
* Native web form validation filtering out junk inputs/spam vectors.
